  • Account Management

    Manage your Syncplicity account, including account owner, administrator access, branding, concurrent access from multiple devices, and VPN connectivity.

  • Reports

    Access the Syncplicity account dashboard as well as detailed Syncplicity reports on utilization and various activities.

  • Policies

    Manage the policies that control how your Syncplicity account is used, including remote wipe, file sharing, file retention and mobile, desktop and web access.

  • Clients

    Manage the mobile and desktop devices used to access the Syncplicity account, which includes device access, remote wipe, and client deployment.

  • Panorama

    Install and manage Syncplicity Panorama to provide secure mobile access to content repositories such as Network Shares, Home Directories and SharePoint.

  • Users & Groups

    Provision and manage user accounts, and organize users into groups for sharing, quota, policy and storage set allocation.

  • StorageVaults

    Install and manage StorageVaults, which allow content storage off-premise in a public cloud, on-premise or hybrid options.

  • Connectors

    Install and configure Enterprise Edition connectors to SharePoint and Documentum.

  • Secure Shared Files

    Protect your Office (Word, Excel, and PowerPoint) and PDF documents. The protection remains in place regardless of where the documents reside.
