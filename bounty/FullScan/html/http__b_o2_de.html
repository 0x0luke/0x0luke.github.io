Wir nehmen das Thema Datenschutz sehr ernst und respektieren Ihre Privatsphäre.

Welche Daten erhebt der Linkshorter b.o2.de?
Wie beim Besuch jeder Internetseite wird neben technischen Daten wie Geräte-Typ oder verwendeter Browser auch Ihre aktuelle IP Adresse an den Shorter-Server übertragen. Vor der Speicherung werden diese Daten vollständig anonymisiert, so dass kein Rückschluss auf einzelne Personen möglich ist. Es findet kein personenbezogenes Tracking anhand dieses Shorter Services statt. Anhand dieser anonymisierten Analysedaten erkennen wir zum Beispiel, wie oft ein Shortlink aufgerufen wurde und können beliebte Shortlinks identifizieren. Es können dabei keine Rückschlüsse auf einzelne Personen durchgeführt werden. Die Daten werden nicht mit anderen Datenquellen im Unternehmem zusammengeführt. Für die mit diesem Shorter-Service gekürzten Ziel-URLs gelten die jeweiligen Nutzungs- und Datenschutzbestimmungen des Betreibers.

Widerspruch gegen Datenerfassung

Sie können die Erfassung/Verarbeitung der anonymisierten Nutzungsdaten durch den Linkshorter b.o2.de verhindern, indem Sie auf folgenden Link klicken:

Durch einen Klick auf diese Schatfläche wird ein Opt-Out-Cookie in Ihrem Browser gesetzt (Gültigkeit des Opt-Out-Cookies: 5 Jahre), das die Erfassung von Daten bei künftigen Aufrufen von b.o2.de -Links verhindert.
Die Analysefunktion wird dann für alle Shortlinks der Domain b.o2.de deaktiviert, solange Sie das Opt-Out Cookie nicht löschen.
