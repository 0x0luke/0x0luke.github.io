This was an interesting question that came up during a discussion at office.

There is initially an NxN matrix with black and white squares.
At each iteration, any white cells that has >=2 bordering black cells also turns black. (border is defined by shared edge only, so each cell has 4 neighbors)
This continues till there can be no more change in the matrix.

The question is,
what is the minimum number of cells that need to be originally colored black so that the entire matrix becomes black eventually. What is the initial configuration of the board?
