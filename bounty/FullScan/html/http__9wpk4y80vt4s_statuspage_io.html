Driver incompatibility identified affecting a small percentage of protected systems Subscribe
Identified - We have identified driver issues in some specific scenarios for systems using the Datto Windows Agent versions 2.0.X and 2.1.X. Symptoms of this issue are: high CPU utilization during backups, a recoverable soft lock-up of virtual machines, and exhibiting 0x00000133 (DPC_WATCHDOG_VIOLATION) BSOD errors.

A fix for these issues is in development and expected to be issued in early August. Please contact Support if you are experiencing these issues or have questions.
