At Knight Lab, we try to write easy-to-use tools with good documentation, but we know that sometimes there are still questions. We are an academic lab, not a business, so we can't guarantee a quick response, but we do what we can to help as time and resources permit.

How to get help

  1. Read the product documentation/help sections: TimelineJS, StoryMapJS, SoundCiteJS, JuxtaposeJS
  2. Search this forum (or browse project sections below)
  3. If you still have questions, ask us in our Zendesk help center or send an email to support@knightlab.zendesk.com*

*Please use Zendesk for your questions -- GitHub and Twitter are not the best way to ask us for help.
