EMEA: Release of Jitterbit Harmony 10.5 on 8 August 2019 Aug 8, 20:00 - Aug 9, 00:00 UTC
Announcing the upcoming release of Jitterbit Harmony version 10.5 (Harmony Portal and its applications) to the EMEA region on 18 July 2019.

In addition, the Sandbox Cloud Agent Group is being upgraded from agent version 10.4 to agent version 10.5.

This release is scheduled to be deployed from 2019-08-08 20:00 UTC through 2019-08-09 00:00 UTC (Thursday, August 8, 2019 from 1:00 pm PDT through 5:00 pm PDT).

Design Studio, Private Agent, and Private API Gateway installers will be available at the conclusion of the release.

There are no planned disruptions or downtime for this release.

If you have any questions, contact support. See the release notes for details.
