Welcome to the Help Center of Bookly - Wordpress #1 booking and appointment scheduling plugin.

Here you will find useful resources, which will help you to install our plugin, master Bookly operations, configure front-end, backed and more. Bookly started in 2014, and now thousands of clients use our solution to sell services online and manage the appointment scheduling routine. We have tons of information to share.

The Intro page is a starting point, which will help new Bookly users learn step-by-step how to install and use Bookly products. For more details, we recommend using search, checking documentation, FAQ, and forum.

Help Center is also the best way to talk to us and offers such communication options like tickets(requires valid support pack), feature requests and forum. Learn more about our support policy.
