                  PayPal's white label mobile wallet solution.

                  Paydiant is now PayPal's White Label Wallet (WLW) platform.

                  The WLW Platform puts the power of PayPal into a physical location that enables in-person mobile payments. Partners use the WLW APIs described here to deploy branded mobile wallet apps that work at the point of purchase at retail, restaurant, fuel site, cash access atm, and other in-person locations.

                  • For recent changes to the documentation, see Release Notes.

                  API Guides

                  The WLW API Guides describe the protocols and methods used to access the WLW Platform through the available WLW Gateways.

                  • B2B Mobile Transaction API
                  • B2B Customer Care

                  The "101" guides provide brief tutorials on how to use these APIs:

                  • B2B Mobile Transaction API 101
                  • B2B Customer Care 101

                  Integration guides

                  Integration guides describe how to add Apple Pay or PayPal into a WLW-powered mobile wallet solution.

                  Campaign Portal (SSP)

                  The WLW Offer Campaign Portal is a web-based tool used to design and deploy offer campaigns for WLW-powered mobile wallets. The Campaign Portal Help files are available here and from the Portal using the Help link on every page.
