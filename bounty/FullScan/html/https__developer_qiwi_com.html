Новая аутентификация


Добро пожаловать снова — теперь для аутентификации не требуется ввод капчи. Одно подтверждение по SMS, и вы готовы обмениваться данными с Кошельком.

А еще у токенов API появились уровни доступа. Например, можете настроить доступ только для просмотра истории и информации профиля, без платежных функций.


Чтобы выпустить и настроить токен, от вас нужно только несколько кликов — попробуйте.
